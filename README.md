# Portfolio Personale

Benvenuti nel repository del mio portfolio personale, realizzato con **React**, **Vite** e **Bootstrap 5**.

---

## 📋 Descrizione

Questo progetto è un portfolio web che presenta le mie competenze, esperienze, progetti e un modulo di contatto. L'obiettivo è mostrare in modo chiaro e professionale il mio percorso come sviluppatrice web full stack, con un design moderno e responsivo.

---

## 🛠 Tecnologie Utilizzate

- **React**: Libreria JavaScript per costruire interfacce utente dinamiche e reattive.
- **Vite**: Build tool moderno e veloce per progetti front-end.
- **Bootstrap 5**: Framework CSS per uno sviluppo rapido di layout responsivi e componenti UI.
- **JavaScript (ES6+)**: Linguaggio principale per la logica applicativa.
- **HTML5 & CSS3**: Struttura e stile del sito.
- **Bootstrap Icons**: Icone vettoriali per arricchire l'interfaccia.

## 🚀 Come Avviare il Progetto in Locale

### Prerequisiti

- Node.js 
- npm o yarn come package manager

### Istruzioni

1. Clona il repository:

   ```bash
   git clone https://github.com/MaDGiiRL/portfolio-sofia
   cd portfolio-react-vite
   ```

2. Installa le dipendenze:

   ```bash
   npm install
   # oppure con yarn
   yarn install
   ```

3. Avvia il server di sviluppo:

   ```bash
   npm run dev
   # oppure con yarn
   yarn dev
   ```

4. Apri il browser e vai su:

   ```
   http://localhost:5173
   ```
